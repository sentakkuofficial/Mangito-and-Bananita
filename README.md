# Day 3 Demo - Mangito & Bananita

Interactive browser demo for Day 3 of a beginner Python course.

## Story
Bananita sacrificed everything so Mangito could go to law school.
Mangito becomes successful, marries Cherita, and forgets her completely.

Now Bananita sees him again for the first time.

## Demo Flow
1. Click RUN
2. Read the story
3. Make your choices
4. Reach one of the endings

## Choices
- approach / avoid
- remind / stayquiet
- beg / leave

Part 1 ends with a cliffhanger.
